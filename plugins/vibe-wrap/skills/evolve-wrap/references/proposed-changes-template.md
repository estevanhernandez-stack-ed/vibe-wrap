## How to apply

1. Read each proposed edit against the pattern it answers. If the pattern is real and the edit is safe, apply it.
2. Apply edits one at a time with a `[y/n]` per item — never bulk-apply.
3. Re-run the affected SKILL or script against a real session to confirm the change holds.
4. Log any applied edit as a decision (the wrap flow's decision-log gate, or directly) so the next `evolve-wrap` cycle sees what changed and why.
